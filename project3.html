Project 3: Moviewer

Moviewer is a web application designed to simplify how users discover, organize, and track movies or TV shows across multiple streaming platforms. The app aggregates information using an existing API, presenting details like availability, IMDb ratings, and streaming locations in a clear, user-friendly format. By allowing users to create personalized playlists, Moviewer offers a streamlined way to manage "watch later" lists, much like Goodreads for movies.

The project was part of the Interaction Programming and the Dynamic Web course. This course focused on developing interaction-intensive web applications across different platforms, equipping us with the skills to build responsive user interfaces and support seamless user interactions.

Our development process included creating a class diagram to structure the code, wireframes for visual guidance, and mapping the user flow to ensure an intuitive experience. We also conducted usability testing to refine the interface and enhance overall user engagement.

Collaborators: Oscar Persson, Sebastian Thunberg, Anna Nilsson
